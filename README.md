
# Милый, любимый, стоящий больших денег: лексемы из "Словаря русских народных говоров"

Интерактивная карта диалектных слов со значениями **«милый, любимый»** и/или **«стоящий больших денег»**, собранных из **Словаря русских народных говоров** (СРНГ).

**Автор:** Радецкая Ева Максимовна (г. Петрозаводск, 2026)

---

## О проекте

Проект представляет собой интерактивную карту России с точками фиксации лексем.  
При нажатии на точку открывается подробное окно с лингвистической информацией (значение, примеры, регион, год фиксации и др.).

Карта построена с помощью библиотеки **Folium** (Leaflet), полностью автономна после генерации и поддерживает **три режима отображения**:

1. Только лексемы в значении **«милый, любимый»**  
2. Только лексемы в значении **«стоящий больших денег»**  
3. Оба значения одновременно (разные цвета)

## Как запустить

### 1. Установка зависимостей

**Вариант А (рекомендуемый) — с виртуальным окружением**

```bash
# 1. Создаём окружение
python3 -m venv venv (или python -m venv venv)

# 2. Активируем
source venv/bin/activate   # Linux/macOS
# или
venv\Scripts\activate      # Windows

# 3. Устанавливаем библиотеки
pip install -r requirements.txt
```

**Вариант Б (простой)**

```bash
pip3 install -r requirements.txt (или pip install -r requirements.txt)
```

### 2. Генерация карты

Запустите скрипт:

```bash
python3 map_script.py (или python map_script.py)
```

После запуска программа предложит выбрать значение лексем:

```
Выберите вариант:
1 — Лексемы в значении «милый, любимый»
2 — Лексемы в значении «стоящий больших денег»
3 — Оба значения
Введите номер варианта (1/2/3):
```

В зависимости от выбора будет создан один из файлов:

| Режим | Выходной файл |
|-------|----------------|
| 1     | `russia_dialect_map_love.html` |
| 2     | `russia_dialect_map_money.html` |
| 3     | `russia_dialect_map_both.html` |

Откройте полученный `.html` в любом браузере.

### Формат входных файлов

Оба Excel-файла должны содержать лист с именем `Лист1` и следующие колонки (названия строго как ниже):

- `ADRESS / PLACE` — локация 
- `LAT`, `LON` — координаты
- `Лексема` — диалектное слово, обнаруженное в СРНГ
- `Источник`, `Помета`, `Формы`, `Варианты ударения, произношения, грамматических форм`, `Значение`, `Оттенки значения`, `Все значения`, `Пример из словаря`, `Устойчивые сочетания`, `Регион фиксации`, `Год фиксации`, `НКРЯ`, `Устойчивые сочетания1` — это дополнительные колонки, которые используются при наличии таковых (если колонки пустые, то они не будут отображаться). 

## Пустые ячейки

- **LAT / LON / Лексема** — при пустом значении точка **не создаётся**
- **Остальные поля** — пустые значения просто **не показываются** в popup-окне  
- Символы `–`, `-`, пробел приравниваются к пустым значениям

Ошибок не возникает, карта генерируется корректно.

## Особенности реализации

- **Цветовая схема:** для каждой уникальной лексемы (или для категории в режиме 3) автоматически подбирается цвет из палитры 24 базовых оттенков, варьируемых по насыщенности, яркости и типу обводки (сплошная/пунктир, цвет заливки/границы). Это гарантирует визуальное различие даже при большом количестве слов.
- **Легенда:** динамически формируется снизу от карты и показывает соответствие цвета и лексемы (или значения).
- **Всплывающее окно:** содержит все непустые поля из исходных данных для выбранной точки.

## Возможные ошибки и решения

| Ошибка | Решение |
|--------|---------|
| `No module named 'pandas'` / `'openpyxl'` | Выполните `pip install -r requirements.txt` |
| `FileNotFoundError: [Errno 2] No such file or directory: 'table love.xlsx'` | Убедитесь, что файлы `table love.xlsx` и `table money.xlsx` находятся в той же папке, что и скрипт |
| `KeyError: 'LAT'` или другая колонка | Проверьте названия столбцов в Excel — они должны в точности соответствовать указанным выше |
| Карта отображается пустой или без точек | Проверьте, что в данных есть ненулевые координаты (LAT, LON) и заполнена колонка `Лексема` |
| Вместо русских букв отображаются непонятные символы (кракозябры) | **Проблема с кодировкой.** Решения:<br>1. Сохраните Excel-файл в формате `.xlsx` (не `.xls`)<br>2. Откройте файл → «Файл» → «Сохранить как» → выберите «Книга Excel (*.xlsx)»<br>3. Если файл уже в `.xlsx`, пересохраните его заново<br>4. Убедитесь, что в HTML-странице указана кодировка UTF-8 (она прописана в скрипте автоматически)<br>5. Откройте готовый `.html` в браузере через «Файл» → «Открыть», а не двойным кликом |
| Карта не открывается в браузере | Используйте современный браузер (Chrome, Firefox, Safari). При локальном открытии иногда блокируются скрипты — попробуйте запустить через простой HTTP-сервер: `python3 -m http.server` |
